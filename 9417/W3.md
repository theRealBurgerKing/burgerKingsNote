# Word2Vec 四个核心概念详解

---

## 一、两种架构

### 🔷 CBOW（Continuous Bag of Words，连续词袋模型）

**思想：** 用**周围词预测中心词**

```
[我, 喜欢, ___, 学习] → 预测 → "深度"
```

- **输入**：上下文窗口内的多个词
- **输出**：预测中间的目标词
- **特点**：对上下文取平均，信息有损失
- **优势**：训练快，对高频词效果好

---

### 🔷 Skip-gram（跳字模型）

**思想：** 用**中心词预测周围词**

```
"深度" → 预测 → [我, 喜欢, ___, 学习]
```

- **输入**：目标词
- **输出**：预测上下文中的每个词
- **特点**：每个中心词产生多个训练样本
- **优势**：对低频词、罕见词效果更好，语义质量更高

| | CBOW | Skip-gram |
|---|---|---|
| 预测方向 | 上下文 → 中心词 | 中心词 → 上下文 |
| 训练速度 | 较快 ✅ | 较慢 |
| 低频词表现 | 一般 | 更好 ✅ |
| 数据量要求 | 较少 | 较多 |

## 二、两种训练优化方法

> Word2Vec 的核心问题：词表很大（几万甚至几十万词），每次更新都要对所有词做 Softmax，计算代价极高。

### 🔶 层次化 Softmax（Hierarchical Softmax）

**思想：** 用一棵**哈夫曼树（Huffman Tree）** 替代 flat softmax

```
         根节点
        /      \
      左子树   右子树
      /  \     /  \
    ...  ...  词A  词B
```

- 把词表中所有词放在**叶子节点**
- 高频词离根更近（路径短），低频词离根更远
- 预测一个词 = 从根到叶子的**一系列二分类**（左/右）
- 计算复杂度：**O(V) → O(log V)**

| 优点 | 缺点 |
|---|---|
| 无需负样本 | 树结构构建复杂 |
| 高频词训练更快 | 对低频词效果略差 |
| 理论上更严谨 | 推理时稍慢 |

---

### 🔶 负采样（Negative Sampling）

**思想：** 不更新全部词表，只更新**一个正样本 + 少量随机负样本**

```
正样本：("深度", "学习")  → label = 1
负样本：("深度", "苹果")  → label = 0
负样本：("深度", "跑步")  → label = 0
... （共 k 个负样本）
```

- 将多分类问题转化为**多个二分类**问题
- 负样本按词频的 **3/4 次方**概率采样（平衡高低频词）
- 每次只更新 **k+1** 个词的参数（k 一般取 5~20）

| 优点 | 缺点 |
|---|---|
| 实现简单 | 负样本是近似，非精确 |
| 训练速度快 ✅ | 需要调 k 值 |
| 实践效果好 | 理论不如 HS 严谨 |

## Gensim

Gensim 是一个 Python 的 NLP 库，专门用来训练和使用词向量、主题模型等无监督语义模型。

核心功能包括：训练 Word2Vec / FastText / Doc2Vec、LDA 主题模型、以及对大规模语料做高效的相似度计算。

这题用它是因为它把 Word2Vec 封装得非常简洁，几行代码就能跑起来，不需要自己手写反向传播，专注于观察不同配置下的训练速度差异就行。

## Brown Corpus

Brown Corpus 是一个经典的英语语料库，1961 年由布朗大学编制，是 NLP 领域第一个电子化的百万词级英语文本集合。

## 使用 Trigram 假设

即预测下一个词时，只看**前两个词**（而不是整个句子历史）：

$$P(w_t \mid w_1, w_2, ..., w_{t-1}) \approx P(w_t \mid w_{t-2}, w_{t-1})$$

例如：`"I love ___"` → 根据 "I" 和 "love" 来预测下一个词的概率。

---

## 加入 Laplace 平滑（加一平滑）

**核心问题**：如果某个 trigram 在训练集里从未出现过，概率会是 0，导致整个句子概率为 0。

Laplace 平滑的解决方法是分子分母各加一个小量：

$$P(w_t \mid w_{t-2}, w_{t-1}) = \frac{C(w_{t-2}, w_{t-1}, w_t) + 1}{C(w_{t-2}, w_{t-1}) + |V|}$$

其中 $|V|$ 是词表大小，确保所有 trigram 概率 > 0。
